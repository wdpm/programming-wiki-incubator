# 生活感悟

- 保持独立是第一原则
- 坚持高质量的输入输出
- 培养加分技能
- 珍惜时间
- 永远不要为了感情放弃自己的事业
- 工作以后别随便跳槽
- 不要那么在意薪资和稳定
- 努力让自己成为一个领域的专业者
- 不管有多大成绩，不要自傲
- 远离同质化圈子，多接触不同的人
- 千万别当面拆前辈的台
- 做好人，但不要做烂好人
- 对任何人都保留百分之二十的自己
- 真诚和善良很重要
- 好好赚钱理财
- 抽空多陪陪父母
- 年轻不是资本，健康才是
- 养成规律的作息，保证充足睡眠
- 学会清空负面情绪，给自己的心情构筑防线，不被外界营销号干扰
- 与自己和解，但不为自己辩解
- 拥有不死的好奇心
- 降低期待，增强钝感力，以稳定心态化解生活压力
- 不要和别人做比较
- 勇敢去做你害怕的事
- 修炼自己是单身时的契机
- 年纪可以大心态不能老
- 朝着自己40岁的生活状态而努力
- 不要拖延，立即行动

## 成长

- 成长就是认识自己、寻找自己、宽容自己和接受自己的过程。
- 能跟自己对话的人，只能是自己。
- 只有和自己对话，才能找到自己内心深处的向往和热爱，才能在面对诸多外在诱惑和干扰时，更好的选择和舍弃。
